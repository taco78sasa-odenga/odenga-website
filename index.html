IT / DX / AI × PROCESS

ITコンサルティング

IT・DX・生成AIと業務プロセスの両面から、経営課題を整理。「そもそも何から着手すべきか」を現場と一緒に描き、実行計画に落とし込みます。

MS365 / POWER PLATFORM

Microsoft 365 導入・内製化

SharePoint・Power Apps・Power Automate・Power BI で、社内ポータルや業務アプリ、ワークフローを構築。運用を社内で回せる状態まで伴走します。

GENERATIVE AI / AGENT

生成AI・AIエージェント構築

ClaudeなどのAIを組み込み、メール処理・書類作成・問い合わせ対応といった定型業務を自動化。安全に運用できる設計から実装まで。

BPR / PROCESS

業務プロセス改革

業務フローの可視化・標準化から着手。ムダと属人化を洗い出し、テクノロジーを載せる“前”の土台づくりを丁寧に行います。

SUBSIDY / 補助金

補助金活用支援

中小企業診断士として、事業計画の設計から申請書類、採択後の交付申請・実行まで一貫サポート。DX投資と補助金を接続します。

ERP / SECURITY

基幹システム・ERP選定支援

パッケージ比較・移行計画から、ゼロトラスト（ZTNA）を含むセキュリティ基盤の検討まで。次期基幹システムの意思決定を支えます。

WHY ODENGA

“技術も、業務も、わかる”こと。

ITだけの専門家でも、業務改善だけの専門家でもない。その両方を、現場の解像度で。

01 / 両輪

技術と業務、
両方がわかる

IT・DX・生成AIの実装力と、実際の業務プロセスへの理解。両方を1人が持つから、「絵に描いた餅」でも「現場を無視した導入」でもない、地に足のついた解を出せます。

02 / 現場起点

理屈でなく、
現場が回る前提で

現場で手を動かしてきた経験がベース。実際に使われ、運用が続く前提で設計する。だから導入したのに使われない、が起きにくい。

03 / つくって残す

提案で終わらず、
社内に残す

納品して終わりにしません。ドキュメント整備と伴走で、御社の担当者が自分たちで運用・改善できる状態まで。中小企業診断士の視点で経営にも接続します。

DOMAINS

これまで携わった領域。

製造業から商社、人材領域まで。業種ごとの商習慣を踏まえた支援を行っています。※ 守秘のため業種カテゴリで記載

D-01

製造業

熱処理・表面処理・化学などの中小メーカー。業務フロー可視化、社内ポータル、基幹システム刷新の検討を支援。

D-02

商社・金属流通

ERP／コンポーザブル基盤の移行計画、輸出業務の自動化、サプライチェーン効率化の企画を支援。

D-03

HR・人材テック

海外連携を含むHRプラットフォーム開発のPdM支援、ベンダー比較、補助金申請の伴走。

D-04

AIシステム / 受入支援

AI受発注・業務システムの要件定義、受入テスト設計、運用・保守設計のドキュメント整備。
